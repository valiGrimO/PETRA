====== 3. PETrA ======

PETRA se donne pour objectif de produire la description géométrale de n'importe quel objet d'étude.

Une description géométrale est une représentation géométrique d'un espace ou d'un objet où le plan, les élévations et les coupes sont mises en relation les uns avec les autres. L'image la plus simple est le patron d'un cube.

Cette partie du protocole s'applique avec le fichier `//2_Blender\20210125_template-PETRA.blend`.
