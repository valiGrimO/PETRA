====== zH. Esquisser les contours ======

===== Définition =====
Le but de cette couche d'information est plutôt esthétique. Elle permet de détecter et de rendre les ruptures importantes pour mieux souligner les silhouettes. 
Le rendu optimal est — dans Inkscape — en mode "produit" sur une autre couche d'information.

===== Mise en oeuvre =====
1. **Onglet 1 :** Masquer la sphère témoin.
2. **Onglet 2 :** Sélectionner le maillage, appliquer le matériau `zH_OBN`.
3. **Onglet 3 :** Connecter `Ouput 1` avec `zH_OBN`. Casser toute autre liaison que celle-ci.
4. **Onglet 3 :** Sélectionner le moteur de rendu EEVEE.
5. **Onglet 3 :** Render all cameras.

Les images sont alors enregistrées dans le dossier à la racine du projet nommé `TEMP`.
