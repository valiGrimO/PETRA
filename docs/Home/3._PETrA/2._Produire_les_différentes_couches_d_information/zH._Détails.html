zH. Détails

Définition

Le but de cette couche d'information est plutôt esthétique à l'instar de zH. OBN. La différence réside dans la qualité des détails rendus :

  • zH. Détails permet de rendre compte de petits reliefs de surface, avec un aspect général plus grisé.
  • zH. OBN permet de rendre compte de ruptures franches.

Le rendu optimal est — dans Inkscape — en mode "produit" sur une autre couche d'information.
