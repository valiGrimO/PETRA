R4. Pointosité

Définition

La pointosité est une approximation du rayon de courbure d’une surface. Cette information est visualisée sous la forme d'un gradient en niveau de gris (convexité vers le blanc ; concavité vers le noir).
Cette information aide à discriminer divers états de surface. En fonction de la résolution du maillage, diverses informations peuvent apparaître.

Mise en oeuvre


  1. Onglet 1 : Masquer la sphère témoin.
  2. Permière passe :
    1. Onglet 2 : Appliquer le matériau `R4_POI`.
    2. Onglet 3 : Connecter `Ouput 1` avec `R4_POI`. Casser toute autre liaison que celle-ci.
    3. Onglet 3 : Sélectionner le moteur de rendu Cycles.
    4. Onglet 3 : Render all cameras.
  3. Deuxième passe :
    1. Onglet 1 : Sélectionner le maillage
    2. Onglet 1 : Appliquer un modificateur `Decimate` : 0.25
    3. Onglet 3 : Vérifier que le noeud terminal soit connecté sur la puce `Cam-##_R5_POI-25pc`
    4. Onglet 3 : Render all cameras.
  4. Troisième passe :
    1. Onglet 1 : Sélectionner le maillage
    2. Onglet 1 : Appliquer un modificateur `Decimate` : 0.25
    3. Onglet 3 : Vérifier que le noeud terminal soit connecté sur la puce `Cam-##_R5_POI-10pc`
    4. Onglet 3 : Render all cameras.

Les images sont alors enregistrées dans le dossier à la racine du projet nommé `TEMP`.

Liste des images produites et leurs caractéristiques

Structure du nom des fichiers générés :

[Nom de la caméra]_[Catégorie d'information]-[Pourcentage de la décimation du maillage].jpg

Liste des fichiers :

  • Cam-##_R4_POI-100pc.jpg : Résultat de la pointosité sur un maillage à haute résolution
  • Cam-##_R4_POI-25pc.jpg : Résultat de la pointosité sur un maillage décimé à 25%
  • Cam-##_R4_POI-10pc.jpg : Résultat de la pointosité sur un maillage décimé à 10%

Espace de réflexion

Définition de la pointosité :

https://docs.blender.org/manual/en/latest/render/shader_nodes/input/geometry.html
An approximation of the curvature of the mesh per vertex. Lighter values indicate convex angles, darker values indicate concave angles. It allows you to do effects like dirt maps and wear-off effects.

Il existe un plugin pour améliorer le rendu de la pointosité : https://blendermarket.com/products/osl-improved-pointiness
Il est partiuclièrement utile pour les modèles peu dense car le noeud de pointosité est dépendant de la taille des polygones. Comme nous utilisons des modèles denses, ce "défaut" n'a pas besoin d'être corrigé.
