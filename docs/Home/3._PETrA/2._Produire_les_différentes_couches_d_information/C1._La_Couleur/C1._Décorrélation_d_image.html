C1. Décorrélation d'image

Mise en oeuvre

Dans ImageJ,

  • Lancer le plugin DStretch.
  • Ouvrir l'image `Cam-##_C1_PRT.png` ou `Cam-##_C1_PRV.png`.
  • Deux images apparaissent ; fermer celle qui n'a pas les réglages DStretch dans sa partie inférieure.
  • Sélectionner la zone sur laquelle la décorrélation doit être opérée (l'arrière-plan blanc atténue les résultats).
  • Appliquer les filtres DStretch tour à tour en enregistrant à chaque fois l'image selon le schéma présenté ci-après.

Note : Avec le développement des scripts envisagés, toutes ces étapes intermédiaires seront automatisées.
