5.1. Actualiser la résolution des images

Blender produit des images possédant une résolution de 72 ppi. C'est un paramètre sur lequel nous ne pouvons pas intervenir dans Blender.

Pour modifier la résolution des images en accord avec le choix réalisé au début du projet de documentation, Converseen permet de tout convertir rapidement.

Mise en oeuvre

  1. Cliquer sur `Add image`
  2. Chercher le répertoire où se situent toutes les images rendues
  3. Sélectionner toutes les images
  4. Dans le panneau latéral,
    • Décocher `Scale image`
    • Cocher `Resolution`
    • Indiquer la résolution obtenue au début du projet de documentation (en X et Y)
    • Cocher `Overwrite All`
  5. Cocher `Check All`
  6. Cliquer sur `Convert`
  7. Attendre la fin de l'opération
